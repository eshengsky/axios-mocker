Axios Mocker Demo with Chrome extension
See demo with local data

Response Data:

This demo will request this Github api: https://api.github.com/repos/eshengsky/axios-mocker

If not installed AxiosMocker extension, it will really request the api and show response data above.

If you want to mock the response, do as following:

  • Add code after axios referenceï¼š 
    if (window.AxiosMocker) {
  new AxiosMocker(axios);
}
  • Install the buit-in Chrome extension AxiosMocker.
  • Open the Chrome dev tool panel (Ctrl+Shift+I), click the AxiosMocker tab.
  • Click button New, type 'api.github.com/repos/eshengsky/axios-mocker' in the request url input box, then press Ctrl+S to save.
  • Refresh the page.
